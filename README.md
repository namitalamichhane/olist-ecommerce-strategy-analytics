# olist-ecommerce-strategy-analytics
End-to-end e-commerce analytics project featuring demand forecasting, customer segmentation, and data remediation for strategic planning.
## Data Notice
The raw Olist datasets are publicly available but are not included in this
repository due to size considerations.

Datasets required to run the notebook:
- olist_customers_dataset.csv
- olist_orders_dataset.csv
- olist_order_items_dataset.csv
- olist_sellers_dataset.csv
- olist_geolocation_dataset.csv

Source: Brazilian Olist E-Commerce Public Dataset
# E-Commerce Strategy & Analytics Project

## Overview
This project analyzes Brazilâ€™s Olist e-commerce marketplace to forecast demand,
segment customers, and improve data reliability for strategic planning.

## Business Objectives
- Establish a reliable monthly demand forecast for procurement planning
- Segment customers based on behavioral patterns to support retention strategies
- Resolve data quality issues impacting reporting and forecasting accuracy

## Methods & Tools
- Python (Pandas, NumPy)
- Time-Series Forecasting (Meta Prophet)
- Customer Segmentation (RFM, K-Means Clustering)
- Data Cleaning & Aggregation
- Visualization (Matplotlib, Power BI)

## Key Results
- Achieved a 37.4% weighted MAPE on monthly demand forecasts
- Segmented 90k+ customers into four actionable behavioral archetypes
- Improved model stability by correcting temporal alignment and aggregation errors

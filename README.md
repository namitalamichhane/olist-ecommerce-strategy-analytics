# olist-ecommerce-strategy-analytics
End-to-end e-commerce analytics project featuring demand forecasting, customer segmentation, and data remediation for strategic planning.
## Data Notice
The raw Olist datasets are publicly available but are not included in this
repository due to size considerations.

Datasets required to run the notebook:
- olist_customers_dataset.csv
- olist_orders_dataset.csv
- olist_order_items_dataset.csv
- olist_sellers_dataset.csv
- olist_geolocation_dataset.csv

Source: Brazilian Olist E-Commerce Public Dataset
